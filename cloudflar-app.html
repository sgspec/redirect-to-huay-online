1.1.1.1 Cloudflare DNS Free

ขั้นตอนการติดตั้ง โดยภาพประกอบ

ช่องทาง Download
1.Googla Play (Android) คลิ๊กที่นี่
2.Appstore ( ios ) คลิ๊กที่นี่

📱 หรือทำตามขั้นตอนนี้วิธีเปิด DNS ด้วยแอป 1.1.1.1

หากทำตามขั้นตอนในรูปภาพแล้ว ข้ามขั้นตอนพวกนี้ได้เลย ไม่จำเป็นต้องทำอะไรเพิ่มเติม

ขั้นตอนที่ 1: ดาวน์โหลดแอปจาก Play Store / App Store

ขั้นตอนที่ 2: เปิดแอปแล้วกด ติดตั้ง (Install)

ขั้นตอนที่ 3: กด Install VPN profile แล้วกด Done

⚠️ จำเป็นต้องกดอนุญาต VPN เพื่อให้ DNS ทำงาน

ขั้นตอนที่ 4: กดเมนู ☰ มุมขวาบน แล้วเข้า Settings

ขั้นตอนที่ 5: เลือกโหมด 1.1.1.1

❌ ห้ามเลือก WARP (เพราะเป็น VPN)

ขั้นตอนที่ 6: กด Done

ขั้นตอนที่ 7: กดปุ่มตรงกลางเพื่อเปิดใช้งาน

✔️ หากขึ้นว่า Connected แปลว่าใช้งานสำเร็จแล้ว

สรุป:

  • DNS ฟรี จากCloudflar ปลอดภัย100% และนิยมใช้ทั่วโลก
  • ช่วยให้เข้าเว็บเร็วขึ้น รวมถึงเว็บที่เข้าไม่ได้บางเครือข่าย การเปิดDNSนี้จะเข้าถึงได้อย่างง่ายดาย
  • ใช้ได้ทั้ง WiFi และเน็ตมือถือ ใช้งานอัตโนมัติไม่ต้องตั้งค่าอะไรเพิ่มเติม
