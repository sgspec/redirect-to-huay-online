Android หรือ PC แก้ไขง่ายๆด้วยวิธีตามตัวอย่าง

ขั้นตอนการตั้งค่า DNS (Google Public DNS)

  1. ไปที่ ตั้งค่า หรือ Settings แล้วเลือกหัวข้อ Wi-Fi
  2. เลือกชื่อ Wi-Fi ที่ใช้งานอยู่ และกดที่เครื่องหมายตามภาพด้านล่างนี้:
  3. ในหัวข้อ การตั้งค่า IP เปลี่ยนจาก DHCP เป็น คงที่
  4. ตั้งค่า DNS ตามตัวอย่างนี้:
    DNS 1 = 8.8.8.8
    DNS 2 = 8.8.4.4 หรือ 1.1.1.1 ก็ได้
  5. กดเครื่องหมายถูกหรือตกลงที่มุมขวาบน เพื่อบันทึกการตั้งค่า

เมื่อเสร็จสิ้นแล้ว ให้กลับเข้าแอปอีกครั้ง และคุณจะสามารถเข้าเว็บไซต์ได้ตามปกติ

การตั้งค่านี้ปลอดภัยและฟรี โดยใช้ DNS จาก Google
ทำให้เข้าเว็บไซต์ได้รวดเร็วขึ้น และลดปัญหาโหลดไม่ขึ้น
