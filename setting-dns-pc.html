✅ วิธีเปลี่ยน DNS ไปใช้ของ Google ใน Windows 10

1. กด Start → พิมพ์ Control Panel → เข้าไป 2. ไปที่ Network and Internet → Network and Sharing Center 3. กด Change adapter settings 4. คลิกขวาเน็ตที่ใช้ (Wi-Fi หรือ Ethernet) → Properties 5. ดับเบิลคลิก Internet Protocol Version 4 (IPv4) 6. เลือก Use the following DNS server addresses หรือทำตามวิธีด้านล่าง

ขั้นตอนที่ 1: กด ปุ่มStart หรือ Windows เลือกค้นหา → พิมพ์ Control Panel

ไปที่ Network and Internet → Network and Sharing Center

หรือคำว่า อาจแตกต่างกัน

เลือก Open Network & Internet settings

ขั้นตอนที่ 2: ไปที่การตั้งค่าอะแดปเตอร์

เลื่อนลงและคลิก Change adapter options

ขั้นตอนที่ 3: คลิกขวาเลือก Properties

เลือกอะแดปเตอร์ที่ใช้งานอยู่ เช่น Wi-Fi หรือ Ethernet

คลิกขวา แล้วเลือก Properties

ขั้นตอนที่ 4: ตั้งค่า DNS

ดับเบิ้ลคลิกที่ Internet Protocol Version 4 (TCP/IPv4)

เลือก Use the following DNS server addresses

Preferred DNS server: 8.8.8.8
Alternate DNS server: 8.8.4.4

ขั้นตอนที่ 5: ยืนยันและรีเฟรช

กด OK แล้วปิดหน้าต่างทั้งหมด

กลับเข้าเว็บไซต์หรือแอพได้ตามปกติ

ค่านี้สามารถตั้งไว้ได้เลยสามารถใช้งานได้ทุกเว็บไซต์ปลอดภัยและมีประสิทธิภาพขึ้นทุกเว็บไซต์
